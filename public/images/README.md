## Placeholders Temporales

Actualmente, los servicios usan URLs de placeholder de https://placehold.co.
Estas se mostrarán como imágenes grises con el texto del servicio.

## Reemplazar con Imágenes Reales

1. Agrega las imágenes reales en las carpetas correspondientes
2. Las imágenes deben ser:
   - **Formato**: JPG o WebP
   - **Tamaño recomendado**: 1200x800px para servicios, 800x600px para galería
   - **Peso**: Máximo 500KB por imagen (optimizadas)
3. Mantén los nombres de archivo que se referencian en los archivos .md de servicios

## Optimización de Imágenes

Antes de subir las imágenes, optimízalas usando:
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- O cualquier herramienta de compresión de imágenes

Esto mejorará significativamente la velocidad de carga del sitio.
